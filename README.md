Tourism Guide to Goa

This project is a responsive and interactive tourism website designed to showcase the charm and attractions of Goa. It provides users with an engaging overview of the destination, including popular tourist spots, travel inspiration, and a simple booking interface. The website focuses on delivering a clean user experience with visually appealing design elements and smooth interactivity.

The homepage features a large hero banner that captures the essence of Goa, followed by sections that introduce the destination and highlight key attractions such as Baga Beach, Fort Aguada, and Dudhsagar Falls. Each attraction is presented using a card-based layout with images and brief descriptions, making the content easy to explore.

A dedicated booking section allows users to enter their travel details through a form. The form includes client-side validation to ensure that required fields are filled correctly, including proper phone number formatting. User feedback is enhanced through toast notifications, which provide clear and immediate messages during form submission.

To improve usability, the website includes a dark mode feature that can be toggled by the user. This preference is saved using localStorage, ensuring that the selected theme persists across sessions. Additionally, a back-to-top button appears when scrolling, allowing for quick navigation to the top of the page.

The project also includes links to external hotel booking platforms such as Booking.com, Agoda, and Airbnb, giving users convenient access to accommodation options. The layout is responsive and adapts to different screen sizes, making it suitable for both desktop and mobile devices.

Features
Responsive layout with modern design
Hero banner with high-quality imagery
Attractions displayed using card components
Interactive booking form with validation
Toast notifications for real-time feedback
Dark mode toggle with saved preferences
Back-to-top button for improved navigation
External links for hotel booking

Technologies Used
HTML5 for structuring the content
CSS3 for styling, layout, and responsiveness
JavaScript for interactivity, DOM manipulation, and validation

Purpose

This project demonstrates fundamental front-end development skills, including layout design, responsive styling, and interactive features using JavaScript. It is suitable for beginners and can be extended with backend integration, API-based booking systems, or additional UI enhancements.
